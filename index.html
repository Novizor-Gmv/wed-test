Тег img

предназначен для вставки изображений на страницу - у него есть два обязательных атребута src="" и alt=""

src="" - это путь к картинки
alt="" - Это текст если картинка по какой либо причине не смогла загрузиться
И несколько атрибутов необязательных:
  • align - Определяет как рисунок будет выравниваться по краю и способ обтекания текстом.
  • border - Толщина рамки вокруг изображения.
  • hspace - Горизонтальный отступ от изображения до окружающего контента.
  • ismap - Говорит браузеру, что картинка является серверной картой-изображением.
  • longdesc - Указывает адрес документа, где содержится аннотация к картинке.
  • lowsrc - Адрес изображения низкого качества.
  • vspace - отступ от изображения до окружающего контента.
  • width - Ширина изображения.
  • height - Высота изображения.
  • usemap - Ссылка на тег , содержащий координаты для клиентской карты-изображения.Также для этого тега доступны универсальные атрибуты и события.

Form - Оболочка формы - обязательный атребут тега form ( action="" ) определяет куда полетят данные, некие данные сервера

Частые атрибуты Тега Form
  • action="" - Адрес сервера
  • method="" - Отвечает за метод передачи данных и может иметь два значения
    1. ="get" Который используется по умолчанию - Для малого передачи фалов
    2. ="post" используется для большой информации
  • enctype
  • Value - значение данных списка заполнения, которое будет отправлено на сервер или прочитано с помощью скриптов.

но этим занимаются бекенд разработчики и пока мы ставим просто заглушку #

А нас интересует интерективные элементы Form, отвечающие за отправку данных

input Тег является одним из разносторонних элементов формы и позволяет создавать разные элементы интерфейса и обеспечить взаимодействие с пользователем. Главным образом input предназначен для создания текстовых полей, различных кнопок, переключателей и флажков.

  • type= - главный атрибут данного тега , именно он определяет метод приема данных и разберем его знаечния
    • type="text" - ввод типа данных текст
    • type="password" ввод типа данных как пороль тект будет скрыт
    • type="radio" - выбор данных как напрмиере нажать потверждения чего либо, выбор из одного варианта круглой кнопки
    • type="checkbox" - выбор данных квадратной кнопки и галочек
    • Type="file" - выбр файла для загрузки и следовательно будет отправлен вмесет сформой
  • mane="" - информируемые данные, имя поля
  • value="" - значение данных для отправки на сервер
  • placeholder="" - полупрозрачный текст который пропадет при вводе данных
  • checked - выставляет выбор radio по умалчанию

Если поле ввода предусматривает ввод цифр, особенно телефоныый номер, то необходимо использользовать значение атрибута type со значением tel ( type="tel") это переключит способ ввода на телефоне в цифровой формат

Все поля формы должы быть распаложенны внутри тега form

Тег Textarea - форма ввода

Поле < textarea > представляет собой элемент формы для создания области, в которую можно вводить несколько строк текста.

Атребуты тега textarea

  • name - задает имя поля
  • readonly -устанавливает, что поле не может изменяться пользователем. только для чтения
  • cols - Ширина поля в символах
  • rows - Высота поля в строках текста

Парные теги select и вложенный в него тег option

Тег option определяет отдельные пункты списка, создаваемого с помощью контейнера < select>. Ширина списка определяется самым широким текстом, указанным в теге < option>, а также может изменяться с помощью стилей. Если планируется отправлять данные списка на сервер, то требуется поместить элемент < select > внутрь формы. Это также необходимо, когда к данным списка идет обращение через скрипты.

Атрибут selected ставит пункт выбора по умолчанию, как тут стоит чесночный соус

Пример. Выберите соус для пиццы
